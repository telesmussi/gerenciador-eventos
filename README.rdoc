== About
	
	Gerenciador de Eventos - Damaceno Produções
	
== Usage

  $ git clone git://github.com/lgfaria/eventos.git
  $ cd eventos
  $ bundle install # to make sure you have the Paperclip gem and correct version of Rails
  $ rake db:migrate
  $ rails server

And go to http://localhost:3000/
